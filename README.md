# Game-Compatibility-Pro
Verificador de compatibilidade de hardware integrado com API da steam. Desenvolvido totalmente em python com interface moderna.

Olá! Este é o **Game-Compatibility-Pro**, o meu primeiro projeto oficial no mundo da programação. Sou iniciante e desenvolvi esta ferramenta em cerca de **2 meses**, unindo o meu aprendizado de Python com a minha paixão por hardware e jogos.

## O que ele faz?
O programa ajuda a verificar se o seu PC aguenta rodar qualquer jogo da Steam. 
* Ele identifica o seu **Processador**, **Placa de Vídeo**, **Quantidade de Memória RAM** e **Armazenamento(HD/SSD)**.
* Consulta os requisitos reais na **API da Steam**.
* Mostra um medidor (Gauge) visual com a porcentagem de compatibilidade.

## Tecnologias
* **Linguagem:** Python (Desenvolvido inteiramente nesta linguagem).
* **Interface:** `ttkbootstrap` (Para o visual moderno e Dark Mode).
* **Executável:** Gerado com `PyInstaller`.

* **Disponível como Executável:** Você pode baixar a versão pronta para uso (.exe) na seção de **Releases** à direita, sem precisar instalar o Python!
